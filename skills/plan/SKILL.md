## When To Invoke

Use `/plan` when the user has an unclear assignment, broad request, ambiguous deliverable, or uncertain scope.

## Load Methods

Read and apply:

- `methodologies/manager-alignment.md`
- `methodologies/personal-operating-system.md`

## Workflow

1. Restate the original ask.
2. Separate facts, assumptions, unknowns, and out-of-scope items.
3. Define the desired output and quality bar.
4. Break work into steps with owners, deadlines, deliverables, and dependencies.
5. Write a short confirmation message for the manager or requester.
