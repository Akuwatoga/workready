## When To Invoke

Use `/onboard` when the user is new to a company, team, internship, project, rotation, or manager relationship.

## Load Methods

Read and apply:

- `methodologies/first-90-days.md`
- `methodologies/manager-alignment.md`
- `methodologies/stakeholder-mapping.md`

## Inputs To Ask For Or Infer

- Role title and function
- Team/company context
- Manager expectations
- Current assignments
- People met so far
- Meetings already scheduled
- Known documents, dashboards, reports, or systems
- Confusing terms, processes, or risks

If information is missing, draft with assumptions and mark unknowns.

## Workflow

1. Diagnose what the user is entering: role, team, expectations, and ambiguity.
2. Create a first-week map of people, projects, systems, and questions.
3. Identify credibility-building early wins that are useful, visible, and low-risk.
4. Prepare manager alignment questions.
5. Produce a short message the user can send to confirm priorities and next steps.
