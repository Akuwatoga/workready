## When To Invoke

Use `/message` when the user needs a professional message that is clear, specific, and action-oriented.

## Load Methods

Read and apply:

- `methodologies/manager-alignment.md`
- `methodologies/status-report.md`

## Workflow

1. Identify audience, channel, purpose, stakes, and desired response.
2. Preserve the user's facts and intent.
3. Make the ask or next action explicit.
4. Offer tone variants when useful.
5. Recommend the best version and explain briefly why.

## Output

### Recommended Version

The version the user should send.

### Concise Version

Short version for chat.

### Collaborative Version

Warmer version for peers or cross-functional partners.

### Formal Version

More structured version for managers, clients, or sensitive topics.

### Notes

Mention assumptions, missing context, or risk in the wording.
