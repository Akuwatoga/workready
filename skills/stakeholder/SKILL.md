## When To Invoke

Use `/stakeholder` when the user needs to understand key people, decision makers, influencers, users, or collaborators.

## Load Methods

Read and apply:

- `methodologies/stakeholder-mapping.md`

## Workflow

1. List stakeholders from user notes and infer likely missing groups.
2. Estimate influence, interest, concerns, and value needs.
3. Group stakeholders by communication priority.
4. Recommend next conversations and questions.
5. Create a communication plan the user can execute.
