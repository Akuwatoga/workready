## When To Invoke

Use `/tasks` when the user needs to organize work across tasks, meetings, decisions, projects, follow-ups, and reviews.

## Load Methods

Read and apply:

- `methodologies/personal-operating-system.md`

## Workflow

1. Capture all work items from user notes.
2. Convert each item into a task with status, project, owner, due date, and next action.
3. Identify waiting items and missing owners.
4. Create a daily and weekly review loop.
5. Provide Markdown that can be stored in Obsidian or any notes app.
