# Usage

## Basic pattern

Paste one command file into an AI assistant with your raw notes.

Example:

```text
Use the update command. Here are my notes:
...
```

## Suggested daily workflow

1. At the start of the day, use `commands/tasks.md`.
2. After meetings, use `commands/meeting.md`.
3. When a task is vague, use `commands/plan.md`.
4. Before manager syncs, use `commands/update.md`.
5. At the end of the week, use `commands/retro.md`.

## Suggested first-month workflow

- Week 1: onboard, stakeholder, tasks
- Week 2: plan, meeting, update
- Week 3: raci, status, datamap
- Week 4: retro, first90, message
