# /first90

Use this command when the user needs a 30/60/90-day plan for a new role, internship, probation period, or internal transfer.

## Goal

Build a staged plan that helps the user learn the business, create early wins, align with the manager, and become reliable.

## Output

### Strategic summary

One paragraph explaining the plan.

### 30/60/90-day plan

| Stage | Goal | Key actions | Deliverables | Success signal |
|---|---|---|---|---|
| Days 1-30 | Learn and map | | | |
| Days 31-60 | Deliver and improve | | | |
| Days 61-90 | Own and scale | | | |

### Manager alignment questions

List questions about expectations, priorities, communication cadence, and success criteria.

### Early wins

| Candidate | Why it matters | Scope | Risk | Next step |
|---|---|---|---|---|
