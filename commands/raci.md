# /raci

Use this command when responsibility, accountability, consultation, or information flow is unclear.

## Goal

Create a RACI matrix and clarify ownership without sounding confrontational.

## Definitions

- Responsible: does the work.
- Accountable: owns the final outcome or decision.
- Consulted: gives input before decisions.
- Informed: receives updates.

## Output

### RACI matrix

| Workstream | Responsible | Accountable | Consulted | Informed | Unclear |
|---|---|---|---|---|---|

### Ownership risks

| Risk | Why it matters | How to clarify |
|---|---|---|

### Clarifying questions

List direct but neutral questions.

### Confirmation message

Write a message to confirm ownership with the manager or project group.
