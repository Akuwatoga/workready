# Manager Sync Workflow

Use this workflow before a recurring manager 1:1 or project sync.

## Prepare

Run:

```text
/tasks
/update
```

Bring:

- completed work with evidence
- current risks
- decisions needed
- next actions
- priority tradeoffs

## If Priorities Are Unclear

Run:

```text
/plan
```

Ask the manager to confirm:

- goal
- scope
- output
- deadline
- quality bar

## If Ownership Is Unclear

Run:

```text
/raci
```

Use the confirmation message to clarify who is responsible, accountable, consulted, and informed.
