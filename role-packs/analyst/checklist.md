# Analyst Readiness Checklists

## Intake Readiness

- [ ] I know the business question.
- [ ] I know who requested the analysis.
- [ ] I know who will use the result.
- [ ] I know what decision this analysis supports.
- [ ] I know the expected deliverable format.
- [ ] I know the deadline.
- [ ] I know which metrics need definitions.
- [ ] I know which data sources are likely needed.
- [ ] I know who owns the data or metric definition.
- [ ] I know the biggest risks or assumptions.
- [ ] I have sent or drafted a confirmation message.

## Metric Readiness

- [ ] Metric name is clear.
- [ ] Business definition is clear.
- [ ] Calculation formula is clear.
- [ ] Time window is clear.
- [ ] Population/filter is clear.
- [ ] Data source is clear.
- [ ] Owner or approver is clear.
- [ ] Known caveats are documented.

## Update Readiness

- [ ] Summary starts with the conclusion or status.
- [ ] Progress includes evidence.
- [ ] Risks are visible early.
- [ ] Blockers have owners.
- [ ] Decisions needed are explicit.
- [ ] Next action has a deadline.

## Promotion Evidence Readiness

- [ ] Each project has business context.
- [ ] Each project has personal contribution.
- [ ] Each project has measurable or observable impact.
- [ ] Feedback is recorded.
- [ ] Lessons learned are recorded.
- [ ] Next-stage growth plan is clear.
